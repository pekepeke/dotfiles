## Terminal

- Ctrl + Shift + C : コピー
- Ctrl + Shift + X : カット
- Ctrl + Shift + V : ペースト
- Shift + Ctrl + T : キーにて新規タブ
- Ctrl + Shift + T : タブを新規で開く
- Ctrl + D : タブを閉じる
- Alt + 番号キー : タブ移動
- Ctrl + {, Ctrl + } : タブ移動


## Nautilus

- Ctrl + L : パスの直接入力


## OS アップグレード

```bash
sudo apt-get -y update; sudo apt-get -y upgrade;
sudo apt-get dist-upgrade;
sudo reboot
sudo do-release-upgrade
```
